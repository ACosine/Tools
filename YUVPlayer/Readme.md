##  YUVPlayer

[toc]

#### 1、两种YUV Player播放器

- yuvplayer1.0
- YUVPlayer1.6

---

#### 2、软件支持

- 支持单帧截图及YUV文件片段的剪切拼接


- 支持WindowsXP/2000/2003/Vista/Win7/Win8等操作系统.


- 支持显示比例的设定.


- 支持播放状态的显示.

---


#### 3、快捷操作

​                    左键双击:常视图<->全屏                        

​    			    空格: 播放<->暂停                          

​                    . (句号): 停止

              方向上键: 暂停                               

​                    方向下键: 播放                            

                 Home键: 调至第一帧

                 End键: 调至最末帧                       

​                    方向左键: 单帧回退                        

​                    方向右键: 单帧步进

                 Ctrl + C: 关闭文件                       

​                    Ctrl + O: 打开文件                        

                 Ctrl + T: 剪切文件

              Ctrl + M: 拼接文件                           

​                    Ctrl + One:半数尺寸                     

​                    Ctrl + Two: 原始尺寸
